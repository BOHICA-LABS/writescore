## Atlassian Environment Variables âšª

This ruleset uses environment variables from the `.env` file:

* **JIRA_PROJECTS**: Projects that these rules apply to
* **PRIMARY_PROJECT**: The main project for most operations
* **CONFLUENCE_SPACES**: Spaces used for documentation
* **PRIMARY_SPACE**: The default space for documentation
* **DEFAULT_COMPONENTS**: Standard components to use in tickets
* **DEFAULT_LABELS**: Standard labels to apply to all tickets
* **DEFAULT_PRIORITY**: Default priority level for tickets
* **DEFAULT_ASSIGNEE**: Default assignee for tickets
* **PRODUCT_DISCOVERY_PROJECT**: Project for product discovery ideas
* **RISK_REGISTER_PROJECT**: Project for the risk register
</environment_variables>
